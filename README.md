# 📁 File Organizer (Python)

A simple Python automation tool that organizes files into folders based on their file type (documents, images, videos, etc.).

---

## 🚀 Features

* Automatically sorts files by extension
* Creates folders if they do not exist
* Supports common file types
* Moves unknown files into an **"Others"** folder

---

## 📂 Supported File Types

* **Documents:** `.pdf`, `.docx`, `.txt`
* **Images:** `.jpg`, `.jpeg`, `.png`
* **Videos:** `.mp4`
* **Archives:** `.zip`
* **Programs:** `.exe`

---

## 🛠️ How to Use

### ▶ Option 1: Input (Recommended)

Run the script:

```bash
python organizer.py
```

Then enter the folder path you want to organize.

---

### ▶ Option 2: Command Line

```bash
python organizer.py "C:\Your\Folder\Path"
```

---

## 📌 Example

### Before

```
report.pdf
photo.jpg
video.mp4
```

### After

```
Documents/report.pdf
Images/photo.jpg
Videos/video.mp4
```
